Hmac

generate HMAC from a string (VARCHAR) and the given password

command:shash_hmac('outfmt','algorithm_name','string','password')
parameters: outfmt - a type of the output format for the generated key :: hex: hexadecimal string; raw: raw (binary) string; base64: binary base64 encoded string
algorithm_name - the name of the hash algorithm: MD5, SHA1, ...
string - a string you want to hash
password - a hmac password
result:This returns HMAC string. Otherwise, empty string is returned on failure.
example:select first 1 shash_hmac('base64','SHA1',mycol,'foobar'),mycol from mytable; = "Hello world", "BSD+81FW3Y22Hxl+LT8oXdm1UGo="
select first 1 shash_hmac('hex','SHA1',mycol,'foobar'),mycol from mytable; = "Hello world", "0520fef35156dd8db61f197e2d3f285dd9b5506a"


generate HMAC from a BLOB data and the given password

command:shash_hmac_blob('outfmt','algorithm_name','blob_data','password')
parameters: outfmt - a type of the output format for the generated key :: hex: hexadecimal string; raw: raw (binary) string; base64: binary base64 encoded string
algorithm_name - the name of the hash algorithm: MD5, SHA1, ...
blob_data - a BLOB data you want to hash
password - a hmac password
result:This returns HMAC string. Otherwise, empty string is returned on failure.
example:myblobcol = "Hello world" ; select first 1 shash_hmac('base64','SHA1',myblobcol,'foobar') from myblobtable; = "BSD+81FW3Y22Hxl+LT8oXdm1UGo="
myblobcol = "Hello world" ; select first 1 shash_hmac('hex','SHA1',myblobcol,'foobar') from myblobtable; = "0520fef35156dd8db61f197e2d3f285dd9b5506a"
