fb_shash 1.0

Richard Kotal, info@waplikace.cz, www.aolserver.cz

The UDF module implements hash and hmac interface to openssl library (http://www.openssl.org).
The module was tested with 0.9.7 version of openssl library.

fb_shash has three categories of the commands:
  • hash UDF - create hash string
  • hmac UDF - create hmac string

Small samples: 
select first 1 shash_keygen('hex','MCRYPT',mycol,'MD5','',10,'',0),mycol from mytable;
expected result: "Hello world", "3e25960a79dbc69b674c" 

select first 1 shash_hash('hex','MD5',mycol),mycol from mytable;
expected result: "Hello world", "3e25960a79dbc69b674cd4ec67a72c62" 

select first 1 shash_hmac('base64','MD5',mycol,'password'),mycol from mytable;
expected result: "Hello world", "MyPQK3X37QIxu6NfSOK68g=="
