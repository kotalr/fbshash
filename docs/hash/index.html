Hash

generate HASH from a string (VARCHAR)

command:shash_hash('outfmt','algorithm_name','string')
parameters: outfmt - a type of the output format for the generated key :: hex: hexadecimal string; raw: raw (binary) string; base64: binary base64 encoded string
algorithm_name - the name of the hash algorithm: MD5, SHA1, ...
string - a string you want to hash
result:This returns HASH string. Otherwise, empty string is returned on failure.
example:select first 1 shash_hash('base64','SHA1',mycol),mycol from mytable; = "Hello world", "e1AsOh9IyGCa4hLN+2Od7jlnP14="
select first 1 shash_hash('hex','SHA1',mycol),mycol from mytable; = "Hello world", "7b502c3a1f48c8609ae212cdfb639dee39673f5e"


generate HASH from a BLOB data

command:shash_hash_blob('outfmt','algorithm_name','blob_data')
parameters: outfmt - a type of the output format for the generated key :: hex: hexadecimal string; raw: raw (binary) string; base64: binary base64 encoded string
algorithm_name - the name of the hash algorithm: MD5, SHA1, ...
blob_data - a BLOB data you want to hash
result:This returns HASH string. Otherwise, empty string is returned on failure.
example:myblobcol = "Hello world" ; select first 1 shash_hash('base64','SHA1',myblobcol) from myblobtable; = "e1AsOh9IyGCa4hLN+2Od7jlnP14="
myblobcol = "Hello world" ; select first 1 shash_hash('hex','SHA1',myblobcol) from myblobtable; = "7b502c3a1f48c8609ae212cdfb639dee39673f5e"
